# HT's Music Downloader v4
一个简单的Material Design 3的音乐下载器  
主要支持酷狗源下载，其他源有空可能会加（懒）  
支持根据歌曲名称搜索、酷狗码批量下载、哈希值批量下载等功能  
但是并不具备付费歌曲解锁等功能（我也做不出来）  


Power By Python|Flutter|Flet
